Ways To Win

  1. Score of game at the end of the 1st quarter = $700
  2. Add 1 point to each team's 1st quarter score: (Ex: A score of 7-0 now becomes 8-1) = $450
  3. Score of game at half time = $1,000
  4. Add 1 point to each team's half time score = $450
  5. The box below the real half time scores = $450 (if the winning box happens to be in the bottom row the prize is paid to the box at the top of the same column)
  6. The box above the real half time scores = $450 (if the winning box happens to be in the top row the prize is paid to the box at the bottom of the same column)
  7. Score of the game at the end of the 3rd quarter = $700
  8. Add 1 point to each team's 3rd quarter score = $450
  9. Final score at the end of the game. No 4th quarter prize if the game is tied and goes into overtime. PRIZE IS ONLY GIVEN TO THE FINAL SCORE = $3,000
  10. The box above the final score = $450
  11. The box below the final score = $450
  12. Add 1 point to each team's final score = $450
  13. Add up the final score (Ex: If the final score is 28-14, 28+14=42. The first number is the AFC, second number is NFC) = $500
  14. Add a touchdown to each teams final score (Ex: 28-14 now becomes 34-20. Just add 6 point to each team) = $500
